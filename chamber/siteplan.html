Explore Iceland Site Plan

Site Name

Explore Iceland. This name is direct, descriptive, and memorable, clearly communicating the site's purpose as a travel guide for the country.

Site Purpose

The "Explore Iceland" website is a three-page site designed to be a comprehensive resource for travelers. The project will showcase dynamic and responsive design, accessibility best practices, and the use of modern JavaScript functionality. Content will be driven by a **local JSON file** for attractions and a **remote API** for live weather data, fulfilling key course objectives while building an attractive and functional travel guide.

Scenarios

These questions represent the target audience's needs and will drive the site's content.

  • Scenario 1: Itinerary Planning
    A traveler is planning a trip for next summer. They ask, "What are the must-see waterfalls and black sand beaches on the South Coast?"
  • Scenario 2: Weather and Packing
    A visitor wants to know what to pack. They ask, "What is the weather like in October, and what should I pack to stay warm and dry while hiking?"

Color Schema

The color palette is inspired by Iceland's dramatic landscape and flag, ensuring a consistent and appealing design.

  • Primary Color: `#0A2342` (Dark Blue)
    Used for the header background, section headings, and other key design elements.
  • Secondary Color: `#AEC6CF` (Light Blue/Gray)
    Used as an accent for borders, navigation links, and subtle design details.
  • Accent Color: `#E8F1F2` (Off-white)
    Used for the main content background, providing a clean and readable canvas.

Typography

Two distinct fonts will be used to create a clear visual hierarchy and maintain readability.

  • Headings: 'Roboto', sans-serif
    A clean, modern font that will be used for all headings (h1, h2, etc.) to grab user attention. Example: EXPLORE ICELAND
  • Body Text: 'Open Sans', sans-serif
    A highly readable and versatile font that will be used for all paragraph and body text. Example: This is a paragraph of text.
